## About The Project
Who doesn't love pizza!?  I sure love it.  Online pizza ordering is a staple of modern web interaction, so I felt creating one would be a valuable tool in my arsenal for building websites.  This online pizza ordering portal, built entirely in React (and deployed to a live with via Firebase), allows the user to add items to their order with custom toppings, edit items after creating them, and delete items from their order... all in a slick, modern user interface!
